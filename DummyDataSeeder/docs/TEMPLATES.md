# Templates Documentation

This document describes the auto-generated templates for each Document Type complexity level.

---

## Overview

Templates are automatically generated by `DocumentTypeSeeder` when creating Document Types. Each template is designed to render all properties of its corresponding Document Type.

---

## Template Location

Templates are saved to the `Views/` folder with the same alias as the Document Type:
- `Views/testVariantSimple1.cshtml`
- `Views/testVariantMedium1.cshtml`
- `Views/testVariantComplex1.cshtml`
- etc.

---

## Simple Template

**Used by:** `testVariantSimple{n}`, `testInvariantSimple{n}`

### Properties Rendered
| Property | Type | Rendering Method |
|----------|------|-----------------|
| `title` | Textstring | `@Model.Value("title")` |
| `description` | Textarea | `@Model.Value("description")` |
| `isPublished` | True/False | `@Model.Value<bool>("isPublished")` |

### Template Structure
```html
@inherits Umbraco.Cms.Web.Common.Views.UmbracoViewPage
@{
    Layout = null;
}
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>@Model.Name - {DocumentTypeName}</title>
    <style>
        body { font-family: Arial, sans-serif; margin: 20px; }
        .property { margin-bottom: 15px; padding: 10px; border: 1px solid #ddd; }
        .property-label { font-weight: bold; color: #333; }
        .property-value { margin-top: 5px; }
    </style>
</head>
<body>
    <header>
        <h1>@Model.Name</h1>
        <p><em>Template: {alias}</em></p>
    </header>
    <main>
        <article>
            @if (Model.HasValue("title"))
            {
                <div class="property">
                    <div class="property-label">Title:</div>
                    <div class="property-value">@Model.Value("title")</div>
                </div>
            }
            @if (Model.HasValue("description"))
            {
                <div class="property">
                    <div class="property-label">Description:</div>
                    <div class="property-value">@Model.Value("description")</div>
                </div>
            }
            @if (Model.HasProperty("isPublished"))
            {
                <div class="property">
                    <div class="property-label">Is Published:</div>
                    <div class="property-value">@Model.Value<bool>("isPublished")</div>
                </div>
            }
        </article>
    </main>
    <footer>
        <p>Generated by TestData Seeder</p>
    </footer>
</body>
</html>
```

## Medium Template

**Used by:** `testVariantMedium{n}`, `testInvariantMedium{n}`

### Properties Rendered
| Property | Type | Rendering Method |
|----------|------|-----------------|
| `title` | Textstring | `@Model.Value("title")` |
| `subtitle` | Textstring | `@Model.Value("subtitle")` |
| `summary` | Textstring | `@Model.Value("summary")` |
| `relatedContent` | Content Picker | `@Model.Value<IPublishedContent>("relatedContent")` |
| `blocks` | Block List | `@Model.Value<BlockListModel>("blocks")` |

### Template Structure
```html
@inherits Umbraco.Cms.Web.Common.Views.UmbracoViewPage
@using Umbraco.Cms.Core.Models.Blocks
@{
    Layout = null;
}
<!DOCTYPE html>
<html lang="en">
<head>
    <!-- ... styles ... -->
    <style>
        .block-item { margin: 10px 0; padding: 15px; background: #f5f5f5; border-left: 3px solid #007bff; }
        .block-title { font-weight: bold; color: #007bff; }
    </style>
</head>
<body>
    <header>
        <h1>@Model.Name</h1>
    </header>
    <main>
        <article>
            <!-- Text Properties -->
            @if (Model.HasValue("title"))
            {
                <div class="property">
                    <div class="property-label">Title:</div>
                    <div class="property-value">@Model.Value("title")</div>
                </div>
            }
            <!-- ... subtitle, summary ... -->

            <!-- Related Content -->
            @if (Model.HasValue("relatedContent"))
            {
                <div class="property">
                    <div class="property-label">Related Content:</div>
                    <div class="property-value">
                        @{
                            var relatedContent = Model.Value<IPublishedContent>("relatedContent");
                            if (relatedContent != null)
                            {
                                <a href="@relatedContent.Url()">@relatedContent.Name</a>
                            }
                        }
                    </div>
                </div>
            }

            <!-- Block List -->
            @if (Model.HasValue("blocks"))
            {
                <div class="property">
                    <div class="property-label">Blocks:</div>
                    <div class="property-value">
                        @{
                            var blocks = Model.Value<BlockListModel>("blocks");
                            if (blocks != null && blocks.Any())
                            {
                                foreach (var block in blocks)
                                {
                                    <div class="block-item">
                                        <div class="block-title">@block.Content.ContentType.Alias</div>
                                        @foreach (var prop in block.Content.Properties)
                                        {
                                            <div><strong>@prop.Alias:</strong> @block.Content.Value(prop.Alias)</div>
                                        }
                                    </div>
                                }
                            }
                        }
                    </div>
                </div>
            }
        </article>
    </main>
</body>
</html>
```

### Block List Rendering Pattern
```csharp
var blocks = Model.Value<BlockListModel>("propertyAlias");
if (blocks != null && blocks.Any())
{
    foreach (var block in blocks)
    {
        // Access content type
        var alias = block.Content.ContentType.Alias;

        // Iterate all properties dynamically
        foreach (var prop in block.Content.Properties)
        {
            var value = block.Content.Value(prop.Alias);
        }
    }
}
```

## Complex Template

**Used by:** `testVariantComplex{n}`, `testInvariantComplex{n}`

### Properties Rendered

#### Content Section
| Property | Type | Rendering Method |
|----------|------|-----------------|
| `title` | Textstring | `@Model.Value("title")` |
| `subtitle` | Textstring | `@Model.Value("subtitle")` |
| `bodyText` | Textarea | `@Model.Value("bodyText")` |

#### Media Section
| Property | Type | Rendering Method |
|----------|------|-----------------|
| `mainImage` | Media Picker 3 | `@Model.Value<IPublishedContent>("mainImage")` |
| `thumbnailImage` | Media Picker 3 | `@Model.Value<IPublishedContent>("thumbnailImage")` |

#### Relations Section
| Property | Type | Rendering Method |
|----------|------|-----------------|
| `primaryContent` | Content Picker | `@Model.Value<IPublishedContent>("primaryContent")` |
| `secondaryContent` | Content Picker | `@Model.Value<IPublishedContent>("secondaryContent")` |
| `tertiaryContent` | Content Picker | `@Model.Value<IPublishedContent>("tertiaryContent")` |

#### Blocks Section
| Property | Type | Rendering Method |
|----------|------|-----------------|
| `headerBlocks` | Block List | `@Model.Value<BlockListModel>("headerBlocks")` |
| `footerBlocks` | Block List | `@Model.Value<BlockListModel>("footerBlocks")` |
| `mainGrid` | Block Grid | `@Model.Value<BlockGridModel>("mainGrid")` |

### Template Structure
```html
@inherits Umbraco.Cms.Web.Common.Views.UmbracoViewPage
@using Umbraco.Cms.Core.Models.Blocks
@using Umbraco.Cms.Core.Models.PublishedContent
@{
    Layout = null;
}
<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        .block-item { margin: 10px 0; padding: 15px; background: #f5f5f5; border-left: 3px solid #007bff; }
        .block-title { font-weight: bold; color: #007bff; }
        .grid-item { margin: 10px 0; padding: 15px; background: #e8f4e8; border-left: 3px solid #28a745; }
        .grid-title { font-weight: bold; color: #28a745; }
        .media-image { max-width: 300px; height: auto; }
        section { margin-bottom: 30px; }
        h2 { border-bottom: 2px solid #333; padding-bottom: 5px; }
    </style>
</head>
<body>
    <header>
        <h1>@Model.Name</h1>
    </header>
    <main>
        <!-- Content Section -->
        <section>
            <h2>Content</h2>
            <!-- title, subtitle, bodyText -->
        </section>

        <!-- Media Section -->
        <section>
            <h2>Media</h2>
            @if (Model.HasValue("mainImage"))
            {
                <div class="property">
                    <div class="property-label">Main Image:</div>
                    <div class="property-value">
                        @{
                            var mainImage = Model.Value<IPublishedContent>("mainImage");
                            if (mainImage != null)
                            {
                                <img src="@mainImage.Url()" alt="@mainImage.Name" class="media-image" />
                                <p>@mainImage.Name</p>
                            }
                        }
                    </div>
                </div>
            }
            <!-- thumbnailImage similar -->
        </section>

        <!-- Relations Section -->
        <section>
            <h2>Related Content</h2>
            <!-- primaryContent, secondaryContent, tertiaryContent -->
        </section>

        <!-- Block List - Header Blocks -->
        <section>
            <h2>Header Blocks</h2>
            @if (Model.HasValue("headerBlocks"))
            {
                @{
                    var headerBlocks = Model.Value<BlockListModel>("headerBlocks");
                    if (headerBlocks != null && headerBlocks.Any())
                    {
                        foreach (var block in headerBlocks)
                        {
                            <div class="block-item">
                                <div class="block-title">@block.Content.ContentType.Alias</div>
                                @foreach (var prop in block.Content.Properties)
                                {
                                    <div><strong>@prop.Alias:</strong> @block.Content.Value(prop.Alias)</div>
                                }
                            </div>
                        }
                    }
                }
            }
        </section>

        <!-- Block List - Footer Blocks -->
        <section>
            <h2>Footer Blocks</h2>
            <!-- Similar to headerBlocks -->
        </section>

        <!-- Block Grid - Main Grid -->
        <section>
            <h2>Main Grid</h2>
            @if (Model.HasValue("mainGrid"))
            {
                @{
                    var mainGrid = Model.Value<BlockGridModel>("mainGrid");
                    if (mainGrid != null && mainGrid.Any())
                    {
                        foreach (var gridItem in mainGrid)
                        {
                            <div class="grid-item">
                                <div class="grid-title">
                                    @gridItem.Content.ContentType.Alias
                                    (Column Span: @gridItem.ColumnSpan, Row Span: @gridItem.RowSpan)
                                </div>
                                @foreach (var prop in gridItem.Content.Properties)
                                {
                                    <div><strong>@prop.Alias:</strong> @gridItem.Content.Value(prop.Alias)</div>
                                }
                                @if (gridItem.Areas.Any())
                                {
                                    <div style="margin-left: 20px;">
                                        <strong>Areas:</strong>
                                        @foreach (var area in gridItem.Areas)
                                        {
                                            <div style="margin-left: 10px;">
                                                <em>Area: @area.Alias</em>
                                                @foreach (var areaItem in area)
                                                {
                                                    <div class="grid-item">
                                                        <div class="grid-title">@areaItem.Content.ContentType.Alias</div>
                                                        @foreach (var prop in areaItem.Content.Properties)
                                                        {
                                                            <div><strong>@prop.Alias:</strong> @areaItem.Content.Value(prop.Alias)</div>
                                                        }
                                                    </div>
                                                }
                                            </div>
                                        }
                                    </div>
                                }
                            </div>
                        }
                    }
                }
            }
        </section>
    </main>
</body>
</html>
```

## Block Grid Rendering Pattern

```csharp
var mainGrid = Model.Value<BlockGridModel>("mainGrid");
if (mainGrid != null && mainGrid.Any())
{
    foreach (var gridItem in mainGrid)
    {
        // Grid layout info
        var columnSpan = gridItem.ColumnSpan;
        var rowSpan = gridItem.RowSpan;

        // Content type alias
        var alias = gridItem.Content.ContentType.Alias;

        // Iterate properties
        foreach (var prop in gridItem.Content.Properties)
        {
            var value = gridItem.Content.Value(prop.Alias);
        }

        // Handle nested areas
        if (gridItem.Areas.Any())
        {
            foreach (var area in gridItem.Areas)
            {
                var areaAlias = area.Alias;

                foreach (var areaItem in area)
                {
                    // Process area items (same as grid items)
                }
            }
        }
    }
}
```

## CSS Classes Reference

| Class | Description |
|-------|-------------|
| `.property` | Container for a property |
| `.property-label` | Label for property name |
| `.property-value` | Container for property value |
| `.block-item` | Block List item container (blue border) |
| `.block-title` | Block List item title |
| `.grid-item` | Block Grid item container (green border) |
| `.grid-title` | Block Grid item title |
| `.media-image` | Image styling (max-width 300px) |

---

## Customizing Templates

To customize templates after generation:

1. Edit the generated `.cshtml` files in `Views/` folder
2. Or modify `GenerateTemplateContent()` methods in `DocumentTypeSeeder.cs`:
   - `GenerateSimpleTemplateContent()`
   - `GenerateMediumTemplateContent()`
   - `GenerateComplexTemplateContent()`

---

## Using Partial Views for Blocks

For production use, consider creating partial views for blocks:

```html
<!-- In main template -->
@Html.GetBlockListHtml(Model, "blocks")

<!-- Or for Block Grid -->
@await Html.GetBlockGridHtmlAsync(Model, "mainGrid")
```

Then create partial views:
- `Views/Partials/blocklist/Components/testElementSimple1.cshtml`
- `Views/Partials/blockgrid/Components/testElementSimple1.cshtml`

Refer to Umbraco documentation for Block List and Block Grid rendering:
- [Block List Documentation](https://docs.umbraco.com/umbraco-cms/fundamentals/backoffice/property-editors/built-in-umbraco-property-editors/block-editor/block-list-editor)
- [Block Grid Documentation](https://docs.umbraco.com/umbraco-cms/fundamentals/backoffice/property-editors/built-in-umbraco-property-editors/block-editor/block-grid-editor)
